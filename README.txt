视频处理工具使用说明

1. 系统要求：
- Windows 10 或更高版本
- 已安装 ffmpeg（如未安装，请访问：https://ffmpeg.org/download.html）
- 已安装 Montreal Forced Aligner（安装命令：pip install montreal-forced-aligner）

2. 使用步骤：
1) 解压文件夹
2) 运行 VideoProcessor.exe
3) 按照界面提示操作

3. 文件夹说明：
- res/: 存放原始视频
- temp/: 临时文件目录
- training/: 训练数据目录

4. 注意事项：
- 请确保有足够的磁盘空间
- 处理大文件时可能需要等待较长时间
